# Installation

## Maven

```xml

<dependency>
    <groupId>com.qwlabs.doraemon</groupId>
    <artifactId>lang</artifactId>
    <version>0.1.*</version>
    <type>pom</type>
</dependency>
```

## Gradle

```gradle
implementation 'com.qwlabs.doraemon:lang:0.1.*'
```
