Useful java tools

# Installation

## Maven

```xml

<dependency>
    <groupId>com.qwlabs.doraemon</groupId>
    <artifactId>lang</artifactId>
    <version>0.3.*</version>
    <type>pom</type>
</dependency>
```

## Gradle

```gradle
api 'com.qwlabs.doraemon:lang:0.3.*'
```
